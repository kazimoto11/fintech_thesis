# fintech_thesis
The documents are for the thesis of Derick Kazimoto for his minor dissertation for financial technology thesis.

There are four documents in this repo with the following description:

- Minor dissertation - This is the research paper (thesis) for credit scoring the informal sector
- Data engineering notebook.py - This is a python notebook that consist of data engineering and analysis  of the data  before  modelling.
- LR_Final.R - R notebook for logistic regression modelling and analysis
- SVM_Final.R - R notebook for support vector machines modelling and analysis
